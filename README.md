# Portfolio Project

Welcome to my interactive 3D website portfolio. Here you'll find various projects and experiments that showcase my work in a unique way.

## Project Structure
- **index.html**: The main HTML file that serves as the entry point.
- **/css**: Directory for all CSS files.
- **/js**: Directory for all JavaScript files.
- **/assets**: Directory for images, fonts, and other assets.
- **/3d-scenes**: Directory for 3D models and scene files.

## Features
- Multi-layer interactive 3D website
- Foreground content integration
- 3D backgrounds
- Scroll anchors for navigation
- Enhanced user navigation experience through integrated controls.
